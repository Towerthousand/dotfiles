This setup includes:

* vim
 * `.vimrc`
 * `.vim/colors/hybrid` color scheme
 * Using NeoBundle for plugins (must be installed manually)
 * Plugins:
   * a.vim
   * YCM
   * Airline
   * syntastic
   * nerdtree
   * vim-virtualenv
   * qmake-syntax
   * Fugitive
* tmux
 * .tmux.conf
 * Using TPM for plugins (autoinstalled when first executing tmux!)
* gitconfig (aliases, username, etc)
* bash shell prompt `PS1`
